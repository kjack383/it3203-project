References & Resources

The following websites were used to research the history of the Internet, web technologies, and key concepts presented throughout this project. These resources are excellent references for learning more about web development and networking.

  • MDN Web Docs
    https://developer.mozilla.org/
    Comprehensive documentation for HTML, CSS, JavaScript, and other web technologies.

  • W3Schools
    https://www.w3schools.com/
    Tutorials and examples for HTML, CSS, JavaScript, SQL, and many programming languages.

  • Internet Society
    https://www.internetsociety.org/
    Information about Internet standards, governance, and history.

  • CERN – Birth of the Web
    https://home.cern/science/computing/birth-web
    Historical information about the invention of the World Wide Web by Tim Berners-Lee.

  • Kennesaw State University IT 3203
    https://ksuweb.github.io/IT3203/
    Official course website for the IT 3203 Web Development class.

This website was created as a class project for IT 3203 Web Development.
